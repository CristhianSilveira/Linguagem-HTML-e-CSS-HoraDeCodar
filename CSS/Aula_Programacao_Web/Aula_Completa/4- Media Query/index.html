Media Query:

Isso muda o jogo!

Propriedade que controla o que acontece quando temos diferentes resoluções de tela. Para ver o efeito vamos redimensionar nossa tela. Viu só? Isso é o que acontece quando nós redimencionamos a nossa página usando a propriedade CSS das Medias Queries!

Smartphones (600px para baixo)

O padrão para esse tipo é: @media only screen and (max-width: 600px) {...}

Dispositivos um pouco maiores, mas ainda pequenos (Pequenos Tablets e Smartphones + largos, 600px para cima)

O padrão para esse tipo é: @media only screen and (min-width: 600px) {...}

Dispostivos Médios (Tablets deitados, 768px para cima)

O padrão para esse tipo é: @media only screen and (min-width: 768px) {...}

Dispostivos Largos (laptops/desktops, 992px para cima)

O padrão para esse tipo é: @media only screen and (min-width: 992px) {...}

Super Largos (Telas maiores laptops, desktops e até TVs, 1200px para cima)

O padrão para esse tipo é: @media only screen and (min-width: 1200px) {...}
